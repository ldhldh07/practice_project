### 3.2 근태 등록

- **대상**: HR 담당자, 관리자 (또는 직원 본인 셀프 체크)

- **필수 입력**

- 직원 선택 (Employee)
- 날짜
- 출근 시각 (checkIn)

- **선택 입력**

- 퇴근 시각 (checkOut)
- 근태 상태 (수동 지정 가능)
- 비고/사유

- **운영 규칙**

- 같은 날짜에 중복 등록 불가 (employeeId, date 유니크 검증)
- 출근 시각만 입력 시, 퇴근 미입력 상태로 저장 가능

- **구현 참조**

- `attendance-edit` (feature: add flow)
- 다이얼로그 폼 + 시간 선택기
