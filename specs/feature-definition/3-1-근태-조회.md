### 3.1 근태 조회

- **대상**: 전 역할

- **주요 기능**

- 특정 직원의 근태 이력 조회
- 날짜 범위 필터 (시작일~종료일)
- 근태 상태 필터 (present/late/absent/vacation 등)
- 출퇴근 시각 표시
- 비고/사유 표시

- **표시 항목**

- 날짜
- 출근 시각
- 퇴근 시각
- 근태 상태 (status)
- 비고

- **구현 참조**

- 직원 상세 화면 내 근태 목록 섹션
- Attendance 테이블 쿼리 (employeeId, date range)
