### 4.2 부서별 직원 필터

- **대상**: 전 역할

- **주요 기능**

- 부서 트리에서 부서 선택 시, 해당 부서 소속 직원 목록 필터링
- 하위 부서 포함/제외 옵션 (선택 시 하위 부서 직원까지 조회)
- 직원 목록과 부서 트리 연동

- **운영 규칙**

- 부서 미선택 시 전체 직원 표시
- 부서 선택 시 `departmentId` 기준 필터링
- 하위 부서 포함 모드 시, 선택 부서 + 하위 부서의 모든 직원 조회

- **구현 참조**

- `employee-filter` (feature)
- `department-tree` (feature: 선택 상태 연동)
- `selectedDepartmentAtom` (Jotai) + derived atom으로 하위 부서 계산
