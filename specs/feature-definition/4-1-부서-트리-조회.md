### 4.1 부서 트리 조회

- **대상**: 전 역할

- **주요 기능**

- 부서 계층 구조를 트리 형태로 표시
- 상위 부서 → 하위 부서 관계 시각화
- 부서 선택 시 해당 부서 정보 강조
- 확장/축소 (expand/collapse) 기능

- **표시 항목**

- 부서명
- 소속 인원 수 (headCount)
- 상위/하위 관계 표시

- **구현 참조**

- `department-tree` (feature)
- `department-tree.atom.ts` (Jotai 트리 상태 관리)
- 재귀 트리 렌더링 컴포넌트
