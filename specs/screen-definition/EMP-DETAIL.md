### EMP-DETAIL | 직원 상세

- **접근 역할**: 전 역할

- **목적**

선택한 직원의 상세 정보와 근태 이력을 조회

- **주요 기능**

- 직원 기본 정보 표시 (1-2-직원-상세-조회)
  - 프로필 이미지
  - 이름, 이메일, 전화, 직급, 부서, 입사일, 재직 상태
- 근태 이력 조회 (3-1-근태-조회)
  - 최근 30일 또는 선택 기간의 출퇴근 기록
  - 날짜, 출근, 퇴근, 상태, 비고 표시

- **인터랙션**

- 상단 액션 버튼: 직원 수정 (2-2), 근태 등록 (3-2)
- 근태 행 클릭 시 근태 수정 (3-3) 다이얼로그 열기
- 뒤로가기 버튼: 직원 목록 화면 (EMP-MANAGER 또는 EMP-LIST)로 복귀

- **레이아웃**

- 상단: 직원 기본 정보 카드
- 하단: 근태 이력 테이블
- 우측 또는 하단: 액션 바 (수정, 근태 등록 버튼)

- **구성 컴포넌트**

- `EmployeeInfoCard` (기본 정보)
- `AttendanceHistoryTable` (근태 이력)
- `EmployeeActionBar` (수정/근태 등록 버튼)
